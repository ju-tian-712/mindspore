mindspore.np_dtype
==================

.. py:class:: mindspore.np_dtype

    `np_dtype` 扩展了Numpy的数据类型。

    `np_dtype` 的实际路径为 `/mindspore/common/np_dtype.py` 。运行以下命令导入环境：

    .. code-block::

        from mindspore import np_dtype

    - **数值型**

      ==============================================   =============================
      定义                                              描述
      ==============================================   =============================
      ``bfloat16``                                     NumPy 下的 ``bfloat16`` 数据类型。该类型仅用于构造 ``bfloat16`` 类型的Tensor，不保证Numpy下的完整运算能力。仅在Numpy版本1.20.0至1.26.4生效。
      ==============================================   =============================
